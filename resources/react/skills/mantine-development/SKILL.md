# Mantine Development

## When to use this skill

Use when creating or changing UI in a project that declares `@mantine/core`.

## Procedure

1. Match nearby Mantine components, theme tokens, CSS Modules/styles, and the installed major's APIs.
2. Reuse project wrappers and Mantine primitives before introducing Tailwind/CVA or recreating controls.
3. Preserve provider, portal, modal, notification, dates, and form composition.
4. Verify labels, focus, keyboard behavior, disabled/loading state, and responsive layout.
